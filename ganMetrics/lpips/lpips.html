LPIPS-Score



Zu jedem Input wird eine Menge von Ergebnisbildpaaren generiert.
Als Input kann entweder ein Zählerstandlabel hergenommen werden
(LPIPS von Synt.Generator + Cyclegan)
oder ein synthetisches Bild
(LPIPS nur von Cyclegan)



Beim synthetischem Generator gibt es 3 zufällige Faktoren:
1. Vertikaler Versatz von Ziffern
2. Zufällige leichte Rotation des Bildes
3. Zufällige Grauwertfarben
